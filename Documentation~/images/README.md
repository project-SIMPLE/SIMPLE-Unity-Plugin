# Documentation Images

Store documentation screenshots and GIFs in this folder.

Recommended structure:

```text
Documentation~/images/
├── tutorial/
│   ├── 01-package-manager-git-url.png
│   ├── 01-gama-panel-setup-scene.png
│   ├── 03-generate-preview-button.png
│   ├── 03-static-preview-result.png
│   ├── 04-species-settings-row.png
│   ├── 05-runtime-live-agents-hierarchy.png
│   ├── 06-dynamic-color-discrete.png
│   └── ...
├── user-guide/
└── technical/
```

Current tutorial screenshots are stored in:

```text
Documentation~/images/tutorial/
```

Screenshot guidelines:

- Use PNG for static UI screenshots.
- Use GIF or MP4 only when motion is important.
- Crop screenshots to the relevant UI area when possible.
- Hide unrelated personal paths, windows, or credentials.
- Keep filenames lowercase and descriptive.
- Prefer English UI labels in screenshots when documenting the package.
