# Samples

These folders are imported through Unity Package Manager from the package details panel.

The samples target `simple.webplatform` middleware mode. The startup menu uses port `8080` by default and does not support direct GAMA Server mode in this package.

Declared samples:

- `Unity  VR Template`: base template scene.
- `Code Examples`: example scenes for common GAMA and Unity integration flows.
- `Menu`: startup, IP configuration, and end-of-game menu scenes.
- `Templates`: FPS and sky-view starter scenes.

The package manifest controls which folders appear in the Package Manager sample list.
